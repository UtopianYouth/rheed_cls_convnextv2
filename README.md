# 多任务RHEED图像分析方法研究

## 一、分类部分

基于`convnextv2`完成RHEED图像分类，主要为二维层状生长和三维岛状生长监测。

## 二、检测部分

预计不复现DiffusionDet模型部分，受限于源码运行时，环境复杂问题。

## 三、时序预测部分

暂时性猜想。

## 四、实验的一些基础概念

> - **Mixup**: 数据增强，通过线性插值创建虚拟训练样本；
> - **EMA**: 指数移动平均，用于获得更稳定的模型；
> - **AMP**: 自动混合精度训练，加速训练；
> - **Layer Decay**: 对不同层使用不同学习率。
