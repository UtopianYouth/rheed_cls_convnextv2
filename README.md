# 多任务RHEED图像分析方法研究

## 一、分类部分

## 二、检测部分

预计不复现DiffusionDet模型部分，受限于源码运行时，环境复杂问题。

## 三、时序预测部分

暂时性猜想。

## 四、实验的一些基础概念

> - **Mixup**: 数据增强，通过线性插值创建虚拟训练样本；
> - **EMA**: 指数移动平均，用于获得更稳定的模型；
> - **AMP**: 自动混合精度训练，加速训练；
> - **Layer Decay**: 对不同层使用不同学习率。

# 2.5.1 FCMAE 自监督预训练

> 本节用于描述本文在 RHEED 无标注数据上进行 FCMAE（Fully Convolutional Masked Autoencoder）自监督预训练的原理、实现流程与实验现象，并结合 **train reconstruction loss 曲线**与**四联图可视化（Original / Masked / Reconstruction / Residual）**验证模型是否学习到与 RHEED 物理结构一致的表征。  
> **注意**：请将文中【占位符】替换为你自己的实验设置（图号、数据量、训练轮数、优化器等）。

---

## （1）研究动机与任务定义

在分子束外延（MBE）生长过程中，RHEED 图像能够反映表面周期结构、粗糙度变化、相变/重构等关键信息。面向后续的 RHEED 图像分类、条纹/斑点检测与强度震荡时序建模，若仅依赖监督学习往往面临 **标注成本高、跨批次泛化差、过渡态样本难以稳定标注**等问题。为降低对人工标注的依赖并获取更鲁棒的结构表征，本文引入掩码图像建模（Masked Image Modeling, MIM）范式，在 **RHEED 无标注数据**上采用 FCMAE 进行自监督预训练。

给定输入 RHEED 图像 \(x \in \mathbb{R}^{H\times W}\)（灰度图或复制为 3 通道），通过随机掩码生成可见输入 \(x_v\)，送入编码器 \(f_{\theta}(\cdot)\) 提取潜在表示 \(z\)，再通过轻量解码器 \(g_{\phi}(\cdot)\) 重建被遮挡区域，得到重建结果 \(\hat{x}\)。自监督训练不需要任何人工标签，仅通过重建误差驱动编码器学习包含条纹方向、条纹间距、散射斑点空间分布与背景统计等在内的通用结构先验，从而为后续分类任务（以及可扩展的检测、时序预测任务）提供共享骨干网络初始化权重。

---

## （2）掩码建模与 FCMAE 关键思想

MAE 类方法通过“遮挡大比例区域 + 重建不可见部分”迫使模型学习全局结构表征。然而在卷积骨干中，若仅在输入图像上做遮挡而仍使用普通密集卷积，卷积核会跨越 masked 区域与可见区域进行邻域聚合，导致被遮挡信息在特征计算中发生 **信息泄漏（shortcut）**，从而降低 MIM 任务难度，使模型倾向于学到局部平滑补全而非真正的结构理解。

FCMAE 的核心思想是：**在卷积网络的特征计算过程中显式保持掩码约束，使编码器仅在可见位置/可见区域上进行有效计算**，从而避免信息泄漏，提升预训练对结构建模的约束强度。工程实现上可通过稀疏计算（如 submanifold sparse convolution）或其等效的掩码门控方式实现“仅可见区域参与卷积运算”的效果。

本文预训练采用掩码比例 \(r\)（本文实验取 \(r=0.6\)，即遮挡 60% 区域），在较高遮挡率下学习从有限观测恢复 RHEED 结构信息，以获得更具迁移性的骨干表征。

---

## （3）网络结构与训练流程

### 1）编码器（Encoder）

编码器使用卷积骨干（本文采用【ConvNeXt/ConvNeXt V2】的 FCMAE 形式）。输入 \(x_v\) 经 stem 下采样后进入多阶段卷积块，逐级形成多尺度特征表示。为保证掩码约束在分层结构中一致，可在某一尺度生成掩码后再映射到各 stage 的特征分辨率，实现“同一可见性假设”下的分层特征学习。

### 2）解码器（Decoder）

预训练阶段引入轻量解码器形成 **非对称自编码器**：编码器较重、解码器较轻（例如仅使用少量卷积块/较低通道维度）。该设计使主要表征能力集中学习在编码器中，便于后续迁移到分类/检测等下游任务，同时降低预训练计算开销。

### 3）训练流程（Algorithm）

FCMAE 自监督预训练可总结如下：

1. 从 RHEED 无标注数据集 \(\mathcal{D}_u\) 中采样图像 \(x\)，进行基础预处理（尺寸归一、灰度/通道处理、轻量增强等）。  
2. 以掩码比例 \(r\) 生成随机掩码 \(m\in\{0,1\}^{H\times W}\)（或 patch 级掩码）。其中 \(m=1\) 表示 masked 区域，\(m=0\) 表示可见区域。  
3. 得到掩码输入：
   \[
   x_v=(1-m)\odot x
   \]
   即可见区域保留，遮挡区域置为常数/均值/0（本文采用【与实现一致的方式】）。  
4. 编码器提取潜在表示：
   \[
   z=f_{\theta}(x_v)
   \]
5. 解码器输出重建结果：
   \[
   \hat{x}=g_{\phi}(z,m)
   \]
6. 基于 masked 区域重建误差计算损失并更新参数 \(\theta,\phi\)。

---

## （4）损失函数与优化目标

FCMAE 采用 masked 重建损失，典型为仅在 masked 区域上计算 MSE：

\[
\mathcal{L}_{rec}=\frac{1}{|\Omega_m|}\sum_{(i,j)\in \Omega_m}\left\|\hat{x}_{ij}-\tilde{x}_{ij}\right\|_2^2,
\]

其中 \(\Omega_m\) 表示被遮挡区域（或 masked patch 集合），\(\tilde{x}\) 表示重建目标（原图像或经过 patch-wise normalization 的目标；本文与实现保持一致）。该损失确保模型主要学习不可见区域的恢复能力，避免直接复制可见区域导致训练目标被“走捷径”。

## （5）实验结果与收敛性分析（Train loss 曲线）

图【图号：2-xx】展示了 FCMAE 自监督预训练阶段的 train reconstruction loss 曲线。可以观察到，训练初期 loss 约为 **4.x**，随后随迭代显著下降，并在训练后期逐步收敛并稳定在约 **0.9** 左右，整体呈现“快速下降—缓慢收敛—趋于平台”的典型收敛形态。这表明：

1. **训练过程稳定**：loss 随训练推进持续下降，无明显震荡或发散，说明掩码策略与优化设置合理；  
2. **重建误差显著降低**：从 4.x 降至 0.9 并趋于稳定，说明模型逐步具备从可见区域推断不可见区域的能力；  
3. **进入平台期**：后期 loss 下降变缓，表明模型在当前训练轮数与模型容量下达到相对充分的收敛程度，可作为 checkpoint 选择依据。

需要强调的是，重建 loss 的下降仅能说明预训练任务“跑通并收敛”，但不能直接等价于“下游分类必然提升”。因此本文进一步通过重建可视化，从结构恢复与误差空间分布角度验证模型是否学习到与 RHEED 结构相一致的表征。

## （6）重建可视化与结构一致性验证（四联图）

为直观展示 FCMAE 的结构恢复能力，本文对多种典型 RHEED 场景样本进行四联图可视化，如图【图号：2-xx】所示。四联图从左至右依次为：

- **（a）Original**：原始 RHEED 图像 \(x\)；

  \[
  r = |\tilde{x}-x|
  \]
  （可选：仅展示 masked 区域误差 \(r_m=m\odot|\tilde{x}-x|\)），用于定位模型误差集中区域。

### 现象总结与解释（建议按你四联图的样本类型写）

结合图【图号：2-xx】，可归纳出如下规律：

1. **条纹主导样本**：模型可恢复条纹主方向与主要周期结构，残差主要集中在条纹边缘与局部细节区域，说明编码器学习到了与表面周期结构相关的全局几何信息，而非简单平滑补全。  
2. **斑点主导样本**：在散射斑点显著的样本中，重建结果能够保持点状结构的大体空间分布趋势，残差在强散射点附近略高，反映点状高频结构的恢复难度更大，但整体位置与尺度关系保持一致。  
3. **高噪声样本**：在背景噪声较强的样本中，模型倾向于优先恢复稳定的低频结构并抑制随机噪声，残差集中于噪声突变与纹理不连续区域，体现“结构优先”的鲁棒表征倾向。  
4. **过渡态/模糊样本**：在条纹与斑点混合或结构模糊的过渡阶段，残差更为广泛且误差增大，表明该类样本存在结构成分混叠与边界不稳定，属于更难建模的区域。该现象为后续多任务协同建模提供动机：仅依赖单帧静态分类可能难以稳定区分过渡态，而引入检测（局部结构证据）与时序建模（动态演化证据）有望降低不确定性。

综上，四联图从“结构恢复效果”和“误差空间分布”两方面验证了 FCMAE 在 RHEED 无标注数据上学习到的表征具有结构一致性与一定物理可解释性，与 train loss 的收敛趋势相互印证，为后续下游分类微调提供了可靠初始化。

## （7）与下游任务及多任务主线的关系（承上启下）

本文的分类、检测与时序预测任务并非孤立：分类刻画生长阶段/表面状态，检测刻画条纹/斑点等局部形貌证据，时序预测刻画强度震荡与动态演化。FCMAE 自监督预训练所获得的骨干网络权重可作为共享特征空间的初始化，使不同任务在同一“结构先验”基础上构建任务头并进行微调，从而体现多任务协同建模的意义与工程落地价值。后续章节将基于该预训练权重开展分类微调实验，并进一步探讨其在检测与时序任务中的迁移潜力。
